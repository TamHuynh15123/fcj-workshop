### Thành tích tuần 3:

* Thành công tạo tài khoản AWS Free Tier với cảnh báo billing được cấu hình để tránh phát sinh chi phí bất ngờ

* Tìm hiểu các dịch vụ AWS chính cho dự án Coffee Cloud:
  * **Amplify**: Hosting React frontend với CI/CD
  * **Lambda**: Các serverless backend functions
  * **DynamoDB**: Lưu trữ dữ liệu người dùng và đơn hàng
  * **S3**: Lưu trữ hình ảnh sản phẩm và static assets

* Cài đặt và cấu hình AWS CLI với IAM user credentials phù hợp

* Khám phá AWS Management Console và định vị tất cả dịch vụ cần thiết cho dự án

* Tạo S3 bucket đầu tiên có tên 'coffee-cloud-assets' để lưu trữ file dự án

* Hiểu về giới hạn AWS Free Tier và cách monitor usage để không vượt quá mức miễn phí

* Cấu hình các thiết lập bảo mật cơ bản và tìm hiểu IAM best practices
