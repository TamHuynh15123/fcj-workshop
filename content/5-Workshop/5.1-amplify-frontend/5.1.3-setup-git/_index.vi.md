Trong bước này, chúng ta sẽ khởi tạo Git repository và push code lên GitHub.

## 1. Tạo GitHub Repository

1. Click **+** (góc phải trên) → **New repository**
2. Điền thông tin:
   - **Tên Repository**: `Proposal-AWS---Coffe---FE`
   - **Visibility**: **Public**
3. Click **Create repository**
